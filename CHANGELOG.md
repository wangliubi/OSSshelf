# Changelog

All notable changes to this project will be documented in this file.

## [v3.3.0] - 2024-03-24

### Added
- 文件版本控制功能
- 增强 Markdown 文件预览
- 新增 Excel 文件预览
- 后端错误码统一管理

## [v3.2.0] - 2024-03-23

### Added
- 直接创建文件功能
- 文件直链功能
- 分享页面预览功能

### Improved
- 优化移动端排版
- 其他细节优化

## [v3.1.0] - 2024-03-20

### Added
- 支持文件夹生成分享链接
- 支持指定文件夹生成上传链接给无账号人员上传文件
- 支持 Telegram 分片上传
- 存储桶迁移功能

### Improved
- 其他功能细节优化调整

## [v2.1.0] - 2024-03-19

### Added
- Telegram 存储支持

### Improved
- 优化 WebDAV 在 Windows 资源管理器等场景的使用
- 优化其他一系列功能

## [v1.1.0] - 2024-03-17

### Added
- 初始版本发布
